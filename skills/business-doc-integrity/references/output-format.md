# Integrity Output Format

## 1. Integrity summary
- overall status
- major trust gaps
- circulation readiness

## 2. Top blockers
List the 3-5 highest-impact issues first.

## 3. Detailed issue table
Recommended columns:
- issue
- severity
- location
- evidence/conflict
- why it matters
- suggested fix

## 4. Reconciliation checklist
### Must reconcile now
### Should tighten next
### Optional cleanup

## 5. Gate decision
Use one:
- Pass
- Pass with fixes
- Needs revision before circulation
- Not credible enough yet
