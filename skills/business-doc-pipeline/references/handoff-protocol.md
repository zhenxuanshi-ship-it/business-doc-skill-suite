# Handoff Protocol

## discovery -> blueprint
Pass:
- goal
- audience
- scope
- constraints
- assumptions
- open questions

## blueprint -> draft
Pass:
- chosen doc type
- approved outline
- section intent
- missing inputs list

## draft -> review
Pass:
- current draft
- document type
- intended audience
- known assumptions

## review -> revise
Pass:
- overall verdict
- prioritized issue list
- revision roadmap

## revise -> finalize
Pass:
- revised draft
- major changes made
- unresolved low-priority issues
