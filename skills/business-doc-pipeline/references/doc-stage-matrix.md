# Document Stage Matrix

## If the user has only rough material

Examples:
- meeting notes
- chat logs
- half-formed idea
- leadership instruction
- a few bullets

Start at: `discovery`
Then usually move to: `blueprint`

## If the user wants an outline first

Examples:
- 先帮我列结构
- 先出大纲
- help me structure this

Start at: `blueprint`
Then usually move to: `draft`

## If the user clearly wants a first draft

Examples:
- 写一个 PRD
- 写项目建议书
- draft a feasibility report

Start at: `draft`

## If the user already has a draft

Examples:
- 帮我 review
- 看看这份方案有没有问题
- critique this document

Start at: `review`

## If the user has comments or review feedback

Examples:
- 按这些意见修改
- 我收到了评审意见
- revise based on comments

Start at: `revise`

## If the user mainly wants a clean delivery version

Examples:
- 帮我定稿
- 出最终版
- prepare the final version

Start at: `finalize`
